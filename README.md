# Rocketsim

A fun rocket flight simulator built in Python with matplotlib.

![Rocket Launch](https://via.placeholder.com/800x400?text=Rocket+Simulation)  
<img width="1920" height="1080" alt="image" src="https://github.com/user-attachments/assets/b9bf64a4-6201-4c81-ac60-af157526406f" />


## Features

- Realistic one-stage rocket simulation
- Two parachute deployment options
- Air resistance (drag)
- Air density decreasing with altitude
- Gravity variation with height
- Engine efficiency based on air density
- Multiple graphs: height, velocity, acceleration, thrust, etc.
